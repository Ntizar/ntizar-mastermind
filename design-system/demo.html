Core

Liquid Glass

Tres niveles de intensidad, variantes de color y specular highlight. En Chrome se aplica refraccion via SVG filter; en otros navegadores el backdrop-filter: blur actua como fallback visible.

.glass-subtle

Nivel Subtle

Fondo muy ligero. Para secciones, dividers, areas de bajo contraste.

.glass-standard

Nivel Standard

El nivel por defecto. Cards, paneles, drawers. Blur 16px + saturate.

.glass-intense

Nivel Intense

Para modales, heroes, elementos prominentes. Blur 28px maximo.

.glass-blue

Tinte Azul

Tinte azul para destacar elementos primarios o estados activos.

.glass-orange

Tinte Naranja

Tinte naranja para alertas, CTAs secundarios, notificaciones.

.card-gradient-border

Borde Gradiente

Borde con gradiente azul-naranja via mask CSS. Sin SVG, sin JS.

Integracion

Como usar

Copia o importa el CSS. Las variables hacen que todo sea sobreescribible por proyecto.

HTML 
<link rel="stylesheet" href="ntizar.css">

<!-- SVG filters (pegar antes de </body>) -->
<svg class="ntizar-svg-filters">
  <defs>
    <filter id="ntizar-glass-filter">...</filter>
  </defs>
</svg>

<!-- Uso -->
<div class="glass-standard glass-specular">
  Contenido con liquid glass
</div>

<button class="btn btn-primary">CTA</button>
Sobreescribir variables por proyecto 
/* En tu proyecto, despues de importar ntizar.css */
:root {
  /* Cambiar solo lo que necesites */
  --blue-500: #6366f1;       /* morado en lugar de azul */
  --orange-500: #ec4899;     /* rosa en lugar de naranja */
  --radius-lg: 8px;          /* bordes mas cuadrados */
  --glass-blur-md: blur(8px);/* menos blur */
}
